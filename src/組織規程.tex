\article{目的}
この規程は、株式会社科学計算総合研究所 定款に定めるもののほか、株式会社科学計算総合研究所（以下、「当会社」とする）の組織に関し必要な事項を定めるものとする。
%株主総会および代表取締役は定款に記載済のため、それ以外について記述

\article{組織運用の原則}
組織の運用にあたっては、役員及び従業員は、定められた基本的な業務を忠実に遂行するとともに、相互に関連のある業務について重複又は間隙を生ぜしめることなく、関係各部署と十分協調し、その活動が効率的に行われるよう、進んで調整を図らなければならない。

\article{基盤研究部の事務分掌}
基盤研究部の事務分掌は、次のとおりとする。
\begin{enumerate}
	\item 中期計画に基づき研究活動を行うこと
	\item 研究計画及び研究予算等に関すること
	\item 研究活動の進捗管理及び庶務的事項に関すること
	\item 法人が開催する研究会・ワークショップ等の企画、実施及び運営に関すること
	\item 研究業績評価等に関すること
	\item 研究成果等の発信、広報活動に関すること
	\item 競争的資金に関すること
	\item 研究活動において必要となる調査・その他外部委託契約に関すること
	\item その他、研究調整事務全般に関すること
\end{enumerate}

\article{応用開発部の事務分掌}
応用開発部の事務分掌は、次のとおりとする。
\begin{enumerate}
	\item 中期計画に基づき開発活動を行うこと
	\item 開発計画及び開発予算等に関すること
	\item 研究成果を事業化するための開発に関すること
	\item 開発活動の進捗管理及び庶務的事項に関すること
	\item 開発成果等の発信、広報活動に関すること
	\item 開発活動において必要となる調査・その他外部委託契約に関すること
	\item その他、開発調整事務全般に関すること
\end{enumerate}

\article{営業部の事務分掌}
営業部の事務分掌は、次のとおりとする。
\begin{enumerate}
	\item 中期計画に基づき営業活動を行うこと
	\item 営業計画及び営業予算等に関すること
	\item 事業に関する営業活動、広報活動に関すること
	\item 営業活動の進捗管理及び庶務的事項に関すること
	\item 営業活動において必要となる調査・その他外部委託契約に関すること
	\item その他、営業調整事務全般に関すること
\end{enumerate}

\article{総務部の事務分掌}
総務部の事務分掌は、次のとおりとする。
\begin{enumerate}
	\item 中期計画及び年度計画に関すること
	\item 当会社の財務管理、予算執行に関すること
	\item 当会社における内部統制及び監査に関すること
	\item 当会社における知的財産に関すること
	\item 当会社における人事、採用及び労務に関すること
	\item 当会社における情報システムの開発及び維持管理に関すること
	\item 代表取締役の企画に関すること
	\item 当会社における法務に関すること
	\item 当会社における組織に関すること
	\item 当会社における物品及び役務等の契約に関すること
	\item 当会社における物品及び財産の管理に関すること
	\item 当会社における文書及び捺印に関すること
	\item 当会社における庶務に関すること
	\item 各号に掲げるもののほか、当会社の業務で他のグループの所掌に属しないもの
\end{enumerate}
